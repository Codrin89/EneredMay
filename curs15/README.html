Variables: are containers for storing data values. --------------------------------------------------------------- Types of data: var string = "This is a string"; var number = 123; var boolean1 = true; var boolean2 = false; var array = ["abc", 2, 7213, true, "6745"]; arrays values are accesed by using an index (e.g. array[4] = "6745") var object = { objectString: "something", objectNumber: 1234, objectBoolean: true, objectArray: [213, "abc", "def"] } object values are accesed by using a literal index ( e.g. object.objectString = object["objectString"] = "something") var x; ---> undefined --------------------------------------------------------------- Operators: Math: ("-", "*", "/", "%", "++", "--" ) Assignment: ("=", "-=", "*=", "/=", "%=") String: ("+") Comparison: ("==", "===", "!=", "!==", ">", "<", ">=", "<=") Logical: ("&&", "||", "!") --------------------------------------------------------------- Functions: is a block of code designed to perform a particular task & is executed when "something" invokes it (calls it). ------------------ FUNCTION PATTERN function name(parameter1, parameter2) { code to be executed } ------------------ FUNCTION PATTERN function myFunction(data1, data2) { console.log(data1); console.log(data2); } myFunction(3213, "something");
